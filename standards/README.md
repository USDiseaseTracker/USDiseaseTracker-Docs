# Data Standards

This directory contains the technical data standard specifications for the US Disease Tracker project.

## Documentation Structure

The data standards documentation is organized as follows:

- **[data-standards.md](data-standards.md)** - Quick reference and navigation guide for all data standards documentation

## Related Documentation

For complete guidance on participating in the project:

- **[Data Submission Guide](../guides/DATA-SUBMISSION-GUIDE.md)** - High-level overview of what to submit, when to submit, and submission requirements
- **[Data Technical Specifications](../guides/DATA-TECHNICAL-SPECS.md)** - Complete technical specifications for data submissions
- **[Data Transfer Guide](../guides/DATA-TRANSFER-GUIDE.md)** - Technical details on how to transfer data to the system
- **[Validation Rules](../guides/VALIDATION.md)** - Validation rules and requirements for data submissions
- **[Pilot Phase Overview](../PILOT-OVERVIEW.md)** - Project background and goals

## Current Standards

This directory now primarily contains the [Data Standards Overview](data-standards.md) which provides a quick reference to all data standards documentation. The detailed technical specifications, validation rules, and submission guides have been moved to the `guides/` directory for better organization.

## Quick Start

For jurisdictions getting started:

1. Review the [Data Submission Guide](../guides/DATA-SUBMISSION-GUIDE.md) for an overview of submission requirements
2. Review the [Data Technical Specifications](../guides/DATA-TECHNICAL-SPECS.md) for detailed field requirements
3. Download the [template file](../examples-and-templates/disease_tracking_report_{state}_{report_date}.csv) and [example data](../examples-and-templates/disease_tracking_report_WA_2025-09-30.csv)
4. Consult the [Data Transfer Guide](../guides/DATA-TRANSFER-GUIDE.md) to select a transfer method
5. Contact the project team through your jurisdiction's liaison
