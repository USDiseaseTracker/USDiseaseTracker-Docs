# Example Data Files

This directory contains example data files that demonstrate compliance with the data standards.

## Available Examples

### Case Report Examples
- `case_report_example.json` - Complete case report following the case reporting standard
- `case_report_minimal.json` - Minimal valid case report with only required fields

### Laboratory Data Examples
- `lab_result_example.json` - Laboratory result data example

## Using Examples

These examples can be used to:
1. Understand the structure of compliant data
2. Test data validation scripts
3. Develop data integration tools
4. Train users on data standards
